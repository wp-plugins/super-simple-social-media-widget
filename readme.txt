== Description ==

The easiest way to add social media icons to any widget area. Includes bloglovin, facebook, google plus, instagram, pinterest, store, tumblr, twitter and youtube. Ideal for fashion blogs and lifestyle blogs.


== Installation ==

Installation can be performed by normal WordPress plugin installation methods.
